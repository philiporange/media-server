media-server · embed.js demo

Video Streaming

Embed video files with this simple code:

<script src="https://<your-domain>/assets/embed.js"
        data-file="BigBuckBunny.mp4"
        data-width="640"
        data-height="360"></script>

Audio Streaming

The same embed code works for audio files:

<script src="https://<your-domain>/assets/embed.js"
        data-file="song.mp3"
        data-width="640"
        data-height="100"></script>

Direct Links

You can also use direct links to the player:

  • Video: https://your-server/index.html?file=video.mp4
  • Audio: https://your-server/index.html?file=audio.mp3

Supported Formats

🎬 Video Formats

  • MP4 (.mp4)
  • MKV (.mkv)
  • MOV (.mov)
  • AVI (.avi)
  • WebM (.webm)
  • FLV (.flv)
  • WMV (.wmv)
  • M4V (.m4v)

🎵 Audio Formats

  • MP3 (.mp3)
  • FLAC (.flac)
  • WAV (.wav)
  • AAC (.aac, .m4a)
  • OGG (.ogg)
  • Opus (.opus)
  • WMA (.wma)

Embed Options

Available data attributes:

  • data-file - Media filename (required)
  • data-width - Player width in pixels
  • data-height - Player height in pixels
  • data-autoplay - Auto-start playback (default: true)
  • data-muted - Start muted (default: false)
  • data-show-poster - Show poster for audio (default: true)
